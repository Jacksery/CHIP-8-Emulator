<h2>🧐 Features</h2>

*   Sound timer utilises SDL2 for audio
*   Adheres to Chip-8 quirks
*   Drag & Drop ROM loading

<h2>🛠️ Setup:</h2>

1. **Install SDL2** (MacOS):

```
brew install sdl2
```

2. **Building**:
```
make
```

3. **Running**:
```
./build/debug/Chip8
```

  
  
<h2>💻 Built with</h2>

*   **SDL2** for graphics, audio and input handling
